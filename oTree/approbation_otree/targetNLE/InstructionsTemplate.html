Dans cette partie, vous devez indiquer la position d’un nombre sur une ligne. Vous devrez faire cet exercice {{ C.NUM_ROUNDS }} fois, avec un nombre différent à chaque fois. La ligne va de 0 à 100, et le curseur est initialement positionné sur 0. Vous disposerez de {{ C.DECISION_TIME }} secondes pour déplacer le curseur, avec la souris, et vous rapprocher de la valeur cible. Si vous n’avez pas répondu à la fin des {{ C.DECISION_TIME }} secondes, le programme considèrera que votre réponse est égale à 0 et passera automatiquement à l’exercice suivant.

Calcul du gain

Le gain dépend de la distance entre la position choisie et le nombre cible. Précisément, si d est la distance entre la position choisie et le nombre cible, le gain est égal à {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x d, avec d = |nombre cible - position choisie|. Illustrons ce calcul à l’aide de trois exemples.

Exemple 1: le nombre cible est 12.2, et vous positionnez le curseur sur la valeur 11.5.
Votre gain est alors égal à {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x |12.2 - 11.5| = 2.96 €.

Exemple 2: le nombre cible est 12.2, et vous positionnez le curseur sur la valeur 55.
Votre gain est alors égal à {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x |12.2 - 55| = 0.86 €.

Exemple 3: le nombre cible est 12.2, et vous positionnez le curseur sur la valeur 90.
Votre gain est alors égal à {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x |12.2 - 90| = -0.89 €. Dans ce cas votre gain est de 0 €.
